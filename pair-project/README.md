# 小学四则运算题目生成器

本项目实现一个用于自动生成小学四则运算练习以及批改答案的命令行程序。核心需求：

- 支持自然数与真分数，运算符 `+ - * /` 以及括号。
- `-r` 控制数值范围（必填），`-n` 控制题目数量，`-e/-a` 用于判分。
- 输出 `Exercises.txt`、`Answers.txt` 和 `Grade.txt`，保证题目唯一性且一次可生成 1 万道题。

> 本次提交仅包含目录结构、模块骨架和 CLI 框架；题目生成与判分逻辑将在后续提交补齐。
